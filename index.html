Sprite Generator Demo

This is an 8-bit pixel art style sprite generator, suitable for avatars. This is an entirely JavaScript implementation based on the techniques used by Lj V. Miranda for the Python/Seagull/Matplotlib-based: Sprites-as-a-Service. As such, this version may be easily run client- or server-side.
